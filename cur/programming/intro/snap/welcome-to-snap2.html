O Snap! é uma linguagem de programção que você pode utilizar para dizer a um computador o que ele deve fazer. Um programa é um conjunto particular de instruções para o computador seguir.

Os programas, na maioria das linguagens de programação, utilizam apenas letras (e pontuação), mas o Snap! é diferente: ele é uma linguagem visual. Ao invés de escrever um programa usando apenas o teclado, você arrastará blocos de comandos e conectará esses blocos uns aos outros.

O Snap! também é diferente de muitas outras linguagens de programação de outro modo— você roda os programas em um navegador de internet como o Firefox ou o Chrome. No momento recomendamos que você utilize o navegador Chrome. O endereço onde você utilizará o Snap! é:

http://snap.berkeley.edu/run
