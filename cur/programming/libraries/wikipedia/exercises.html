Quiz: Understanding the Wikipedia API

1. What is the primary purpose of the Wikipedia API?

A) To retrieve data from Wikipedia such as texts, sections, links, and categories.

Correct!

B) To edit and update existing Wikipedia pages.

Incorrect. The Wikipedia API is mainly for data retrieval, not editing.

C) To create user accounts on Wikipedia.

Incorrect. Account management is not the primary purpose of the Wikipedia API.

2. How do you install the Wikipedia API Python wrapper?

A) pip3 install wikipedia-api

Correct!

B) npm install wikipedia-api

Incorrect. npm is used for Node.js packages, not Python libraries.

C) apt-get install wikipedia-api

Incorrect. apt-get is used for Debian-based Linux distributions, not for installing Python libraries.

Quiz: Predicting the Output of Wikipedia API Code


    import wikipediaapi
    wiki_wiki = wikipediaapi.Wikipedia('english')
    page = wiki_wiki.page('tottenham')
    print("Page - Exists: %s" % page.exists())

What will be the output of the above code?

A) Page - Exists: True

Correct! Since the page 'tottenham' exists on Wikipedia, the output will affirm its existence.

B) Page - Exists: False

Incorrect. If the 'tottenham' page did not exist, this would be the output. However, the page does exist on Wikipedia.

C) Error: Page not found

Incorrect. The Wikipedia API does not throw an error for non-existent pages; it returns False for the 'exists' method.

D) None

Incorrect. The 'exists' method will always return a boolean value, not None.
