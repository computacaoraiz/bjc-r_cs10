É hora de trocar de papéis!

Lembrete: se você não está praticando programação em par, agora é uma boa hora de começar!
Se você estava atuando como o navigator, agora você deve atuar como o driver (e seu parceiro, que era o driver, deve atuar agora como o navigator). Cada parceiro deve trabalhar aproximadamente pelo mesmo tempo em cada papel.

Para saber mais sobre programação em par, visite a página de introdução.
