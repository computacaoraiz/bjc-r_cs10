For these practice questions, we will be using several new datasets. You can click this link to download the cholesterol-and-smoking dataset, and this link to download the music-library dataset. You can practice importing a Table from a .csv file by loading it into a DASIS Snap! file. You will have to create a variable to load these files into.

Once you have loaded the datasets into Snap!, you can begin to answer these questions. To answer each of these questions, you may use any of the DASIS blocks we have covered so far in this assignment, and any mentioned on this page. If you would like to play around with this data and explore other blocks, go for it!

  1. The cholesterol-and-smoking.csv shows cholesterol levels for smokers and ex-smokers from a study. Compare the average cholesterol levels of smokers and non-smokers. What is the difference between these values and what does this suggest about the two populations?
  2. How many participants are there in each population in the cholesterol study? Do you think this might have affected the outcome of this study?
  3. The music-library.csv is a table of data gathered from students on how many songs they have in their music library. Using the music library, find how many students were surveyed and the average number of songs they have.
  4. Use the following block to display the data from the table: that displays the distribution of the number of songs in each student's music library. What trends do you notice about this data? Is there other information you would like to know about this data?

A quick note on errors: if you are using a DASIS block and you receive an error, try to walk through your whole query to ensure you are using each block correctly and providing correct arguments to blocks. Sometimes the best way to debug is to follow the arguments to your different functions and see if one is receiving an argument it shouldn't. Debugging is always part of the programming process, so don't get discouraged if it takes a bit of time to find a bug.
