Tables in DASIS are stored using a format you cannot read very easily when you click on your Table. Instead, to view a Table, we will use the VIEW Table block that allows us to see everything we need.

Note the shape of a Table argument in the VIEW Block. This right-facing arrow shape indicates that we need a Table argument.

    Try it yourself:
    Click on itself, then try using the VIEW Table function.

This is why we have the VIEW Table block.
