Data Science is all about asking questions and answering them with data. Though we will cover many more Table methods in the future, we can already answer meaningful questions about data! Let's take a look at an example of a question we can already answer using the blocks we have talked about thus far.

    Try exploring this question yourself:
    How many total passengers rode in these taxi rides?

It may be easiest to first think about how you would answer this question without any code at all. We could look at the column that stores that data, in this case the passengers column, and add up all the numbers there. Doing it this way, we calculate the answer to be 3 + 3 + 1 + 4 + 2 + 1 + 1 + 1 = 16.
