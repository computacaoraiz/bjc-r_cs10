The LIMIT block is a block we can use to get a new Table with only the first n rows in a Table.

The Table that results from a LIMIT block call has the same number of columns as the original Table, and n rows, where n is the argument you pass in.

The LIMIT block takes in 2 arguments. These arguments are:

  1. A number, n, that represents how many rows you want from the top of the Table. Must be 1 ≤ n ≤ number of rows in original Table
  2. A Table

There is another important visual block that utilizes the LIMIT block.

The block shows you the first 5 rows of a Table using an abstraction of the LIMIT block and the VIEW Table block. This is a great way to see the beginning of a Table quickly without having to create a query!

If the Table you call the block on a Table that has less than 5 rows, you will see the entire Table.
