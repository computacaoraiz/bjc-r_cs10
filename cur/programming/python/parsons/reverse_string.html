Exercise 4: Reverse String

Write the function reverse_string(string) that takes in a string and puts it in the reverse order as shown below: 


>>> reverse_string("Alonzo")
'oznolA'

For this problem, instead of dragging and dropping code in the browser, you will write code free form using a text editor. To do so, open up the starter code and proceed to excercise 4 def reverse_string(string). If you need help, refer to this page or ask a staff member for help!

You'll want to use some sort of loop, along with some of the things that you've learned about strings in this section. You may not use the [::-1] syntax. When you think your function works, you can run the tests we've provided.
